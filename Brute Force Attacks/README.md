# 15 - Brute Force Attacks 💥

## Overview
Brute force attacks attempt to gain access by systematically trying
every possible combination of credentials until the correct one is found.

- Brute force attack types
- Common attack tools
- Prevention mechanisms (MFA, hashing, salting, CAPTCHA)
- Password policy best practices
- Security incident response for brute force
