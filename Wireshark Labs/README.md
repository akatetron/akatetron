# 03 - Wireshark Labs 🦈

## Overview
Wireshark is an open-source network protocol analyzer used to capture
and inspect data packets in real time or from saved capture files.

## Labs in this folder

| Lab | Description |
|---|---|
| [Wireshark Packet Analysis of Sample logs ](https://github.com/akatetron/akatetron/blob/main/Wireshark%20Labs/Analyze%20your%20%20packet%20with%20Wireshark.docx) | Filtering and inspecting packets with Wireshark |
|[How to Read ](https://github.com/akatetron/akatetron/blob/main/Wireshark%20Labs/Copy%20of%20How%20to%20read%20a%20Wireshark%20TCP_HTTP%20log.docx)| How to Read Logs|

## Skills Covered
- Wireshark GUI navigation
- Display filter syntax (ip.addr, ip.src, ip.dst, tcp.port, udp.port)
- TCP/HTTP/DNS/ICMP packet inspection
- Protocol layer analysis (Frame, Ethernet II, IPv4, TCP)
- MAC address filtering
- Payload text searching
