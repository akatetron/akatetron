# 06 - Linux File Permissions 🐧

## Overview
Linux file permissions control who can read, write, and execute files.
Managing permissions correctly is a core part of system security.

## Labs in this folder

| Lab | Description |
|---|---|
| [File Permissions in Linux](./file-permissions-in-linux.md) | Checking and updating file permissions |

## Skills Covered
- `ls -la` command
- Permission string interpretation (rwxrwxrwx)
- `chmod` command usage
- User, group, and other permission management
- Hidden file handling
- Directory permission management
