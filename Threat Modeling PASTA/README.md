# 09 - Threat Modeling: PASTA 🍝

## Overview
PASTA (Process for Attack Simulation and Threat Analysis) is a
risk-centric threat modeling framework with 7 stages.

## Labs in this folder

| Lab | Description |
|---|---|
| [PASTA Worksheet](./pasta-worksheet.md) | Full PASTA threat model for a sneaker company app |

## Skills Covered
- PASTA 7-stage framework
- Business and security objective analysis
- Technical scope definition
- Threat and vulnerability identification
- Attack tree modeling
- Risk analysis and security controls
