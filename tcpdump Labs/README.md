# 04 - tcpdump Labs 💻

## Overview
tcpdump is a command-line packet analyzer used to capture and analyze
network traffic. It is widely used in SOC environments for incident investigation.

## Labs in this folder

| Lab | Description |
|---|---|
| [How to Read tcpdump Traffic Log](./how-to-read-tcpdump-traffic-log.md) | Understanding tcpdump log output |

## Skills Covered
- tcpdump output interpretation
- DNS and HTTP traffic analysis
- TCP flag codes (SYN, ACK, FIN, RST, PSH)
- Port identification
- Malicious traffic identification
