# 12 - Access Control 🔑

## Overview
Access control ensures that only authorized users can access
specific resources, following the principle of least privilege.

## Labs in this folder

| Lab | Description |
|---|---|
| [Access Control Worksheet](./access-control-worksheet.md) | Investigating unauthorized payroll access |
| [Data Leak Worksheet](./data-leak-worksheet.md) | Analyzing a data leak from overpermissioned access |

## Skills Covered
- Access control investigation
- Least privilege principle (NIST SP 800-53 AC-6)
- IAM policy recommendations
- Data leak analysis
- RBAC (Role-Based Access Control)
